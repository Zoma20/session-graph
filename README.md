## 📦 About session-graph

session-graph helps you organize your AI coding sessions. Instead of having your work scattered across tools, this app turns those sessions into a searchable graph. It works with popular AI platforms like Claude Code, ChatGPT, DeepSeek, Grok, and Warp. The app links your coding data with known entities using Wikidata. You can run detailed searches using SPARQL queries, a standard way to explore data graphs.

session-graph uses W3C ontology standards to structure your data. This means your knowledge graph follows accepted rules for how information relates. Even if you are new to AI or coding, the app simplifies managing your session history.
